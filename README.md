 JIRA Analytics Tool

Программное средство для аналитики задач через JIRA REST API.

Программа строит 6 аналитических отчетов по закрытым задачам проекта Apache Kafka:
1. Гистограмма времени в открытом состоянии — сколько времени задачи были открыты
2. Диаграммы распределения по состояниям — время в каждом статусе задачи
3. График заведенных/закрытых задач по дням + накопительный итог
4. Топ-30 пользователей по количеству задач (исполнитель/репортер)
5. Гистограмма затраченного времени на основе залогированного времени
6. Распределение задач по приоритетам



Предварительные требования
- Python 3.8 или выше
- Git


Установка

cd JiraAnalytics

Создание виртуального окружения
python -m venv .venv

Активация виртуального окружения
Для Windows:
.venv\Scripts\activate.bat

Установка зависимостей
pip install -r requirements.txt
